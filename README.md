# Note Web Application
This web application allows users to add, delete and read notes

#### Instructions on how to run the app locally
1.Download/Clone the source code
```bash
git clone repolink
```
2.Install dependencies
```bash
npm install
```
3.Run the application
```bash
node app.js
```



#### Dependencies 
 - express.js
 - pug.js
